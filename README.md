# نظام تعداد الأسر (Family Census)

برنامج تعداد لجمع بيانات الأسر مع دعم **التعرّف الصوتي** و**التعرّف الضوئي على مستندات
الهوية** بالذكاء الصناعي (Anthropic Claude).

يجمع البرنامج لكل فرد الحقول الثمانية: **الاسم، تاريخ الميلاد، النوع، الوظيفة، الجنسية،
رقم الهوية، تاريخ انتهاء صلاحية الهوية، وعلاقته بالأسرة**. يبدأ ببيانات **رب الأسرة**
بالإضافة إلى **رقم الهاتف** و**عدد أفراد الأسرة**، ثم يطلب بيانات كل فرد بناءً على العدد.

## البنية (Architecture)

- **`/Server`** — ASP.NET Core Web API (.NET 8) + EF Core + SQLite.
  - جدول `Person` واحد يضم رب الأسرة والأفراد معًا، مع عمود `IsHead` (هل هو رب الأسرة).
  - `ClaudeService` يستدعي Claude (`claude-opus-4-8`) لاستخراج الحقول عبر Structured
    Outputs؛ الـ OCR عبر Claude Vision. المفتاح يبقى في الخادم فقط.
- **`/client`** — React + Vite + TypeScript + React Router، واجهة عربية RTL.
  - التعرّف الصوتي في المتصفح عبر **Web Speech API** (`ar-SA`) ثم يُرسل النص للخادم للتحليل.
  - **شاشة تسجيل (Logging)** مشتركة تعرض كل نص صوتي/ضوئي خام وكل عملية إضافة/تعديل.

## المتطلبات

- .NET SDK 8.0
- Node.js 18+ و npm
- مفتاح Anthropic: متغيّر البيئة `ANTHROPIC_API_KEY` (لازم لميزات الذكاء الصناعي فقط —
  الإدخال اليدوي يعمل بدونه).

## التشغيل (Development)

### 1) الخادم (Backend)

```bash
cd Server
export ANTHROPIC_API_KEY=sk-ant-...   # اختياري لتفعيل الصوت/التصوير
dotnet run
# يستمع على http://localhost:5080  ( Swagger على /swagger )
```

قاعدة البيانات `census.db` تُنشأ تلقائيًا عند أول تشغيل.

### 2) الواجهة (Frontend)

```bash
cd client
npm install
npm run dev
# افتح http://localhost:5173  (يوجّه /api إلى الخادم على 5080)
```

## نقاط الـ API

| الطريقة | المسار | الوصف |
|--------|--------|-------|
| POST | `/api/ai/voice` | تحليل نص منطوق واستخراج الحقول |
| POST | `/api/ai/ocr` | تحليل صورة هوية (base64) واستخراج الحقول |
| POST | `/api/households` | حفظ الأسرة (الأفراد + سجل العمليات) |
| GET | `/api/households/{id}` | استرجاع أسرة بأفرادها |
| GET | `/api/households/{id}/logs` | سجل العمليات لأسرة |

## تدفّق الاستخدام

1. **رب الأسرة**: تعبئة الحقول (يدويًا/صوتيًا/بالتصوير) + الهاتف + عدد الأفراد.
2. **الأفراد**: صفحة لكل فرد بحسب العدد المُدخل.
3. **المراجعة والحفظ**: جدول يعرض الجميع ثم يُحفظ في قاعدة البيانات.

عند استخدام الصوت أو التصوير: يظهر النص الخام في شاشة الـ Logging، ثم يعبّئ الذكاء الصناعي
الحقول (مع إمكانية تعديل/استبدال القيم الموجودة)، ويُسجَّل سطر لكل عملية إضافة أو تعديل.

> ملاحظة: التعرّف الصوتي يتطلب متصفحًا يدعم Web Speech API (Chrome/Edge) وإذن الميكروفون.
